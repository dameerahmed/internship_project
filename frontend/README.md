# Dameer Frontend

This frontend is a React + Vite app created to work with your FastAPI backend.

## Recommended stack
- React + Vite for a fast SPA
- React Router for route-based auth screens
- Fetch API for login/register calls

## Run locally
1. Install dependencies:
   ```bash
   cd frontend
   npm install
   ```
2. Start the dev server:
   ```bash
   npm run dev
   ```
3. Open http://localhost:3000

## Backend connection
Set the API URL before running:
```bash
export VITE_API_URL=http://127.0.0.1:8000
```
